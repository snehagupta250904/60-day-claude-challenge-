# Day 2 - Prompt Engineering

## Objective
Learn the difference between a simple prompt and an engineered prompt.

## Lazy Prompt
Create an image explaining Prompt Engineering.

## Engineered Prompt
Create a professional infographic explaining Prompt Engineering with definition, types, applications, examples, and best practices.

## Observations
- Engineered prompt produced better results.
- Better structure and design.
- More detailed explanation.
- More professional output.

## Key Learnings
- Specific prompts give better results.
- Context improves AI responses.
- Output quality depends on prompt quality.
